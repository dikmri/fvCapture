# v0.7.2 リリースノート

- 録画停止時にメインウィンドウを表示、最小化解除、フォーカス要求するようにしました。
- 録画後プレビューを専用の `プレビュー` タブへ移動し、録画停止後に自動で表示するようにしました。
- 動画出力後も録画後プレビューを保持し、形式、保存先、出力フレーム範囲を変更して再出力できるようにしました。
- 録画中/一時停止中のアプリアイコン更新を Windows のネイティブアイコンにも反映するようにし、状態バッジを見やすくしました。
- 保存先の既定値をドキュメントフォルダに変更し、空欄や相対パス指定時もドキュメントフォルダ配下へ解決するようにしました。
