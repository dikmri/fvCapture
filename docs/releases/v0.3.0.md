# v0.3.0 リリースノート

- GUIで範囲指定を数値入力ではなく、プレビュー上のドラッグ操作で指定できるようにしました。
- 録画範囲にウィンドウ単位の選択を追加しました。
- 操作ラベルのプレビューを追加し、録画前に見た目を確認できるようにしました。
- 録画開始/停止をF9、一時停止/再開をF10で操作できるようにしました。
- 録画停止後の動画出力をバックグラウンド処理に変更し、アプリが固まったように見えないようプログレス表示を追加しました。
- WindowsでFFmpeg起動時にCUIウィンドウが出ないようにしました。
- 言語設定を画面右上へ移動しました。
- 日本語UIフォントを太めの表示に変更し、任意のフォントファイルを選択できる設定を追加しました。
- カーソル表示のオン/オフを追加しました。
- 操作ラベルの位置、表示時間、フォント風スタイル、キー背景色、文字色、枠線色、マウス色、カーソル色を調整できるようにしました。
