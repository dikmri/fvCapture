# v0.6.0 リリースノート

- 録画停止後に動画プレビューを表示し、先頭/末尾のハンドルで出力範囲を調整できる簡易編集機能を追加しました。
- 録画後プレビューで、選択範囲だけの再生、ループ再生、先頭/末尾から除去するフレーム数指定に対応しました。
- 範囲指定で、画面上のドラッグ選択が完了した時点で範囲選択画面を閉じるようにしました。
- 範囲指定後に、選択した録画範囲のプレビューを録画範囲タブへ表示するようにしました。
- グローバル F9 / F10 で録画開始/停止や一時停止/再開を操作した際に、確認音を鳴らせる設定を追加しました。
- 操作ラベルのプレビュー描画を実際の動画合成処理と揃え、プレビューと出力動画で文字の見た目が一致するようにしました。
- 画面上の操作ラベルプレビューで、閉じるボタンをクリックしやすい位置に表示するよう修正しました。
- MP4 / WebM 出力時に録画範囲の幅または高さが奇数でもエンコードに失敗しないよう、FFmpeg 出力を偶数サイズへ補正するようにしました。
