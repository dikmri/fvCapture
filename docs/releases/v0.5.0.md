# v0.5.0 リリースノート

- 設定項目を録画範囲、操作ラベル、出力、表示のタブに分け、1画面で操作しやすくしました。
- 範囲指定を実際の画面上でドラッグして選べるようにしました。
- ウィンドウ録画で、選択中ウィンドウのプレビュー画像を確認できるようにしました。
- 操作ラベルを実際の画面上に重ねてプレビューできるようにしました。
- F9の録画開始/停止、F10の一時停止/再開をグローバルショートカットとして登録し、fvCaptureが非アクティブでも操作できるようにしました。
- 保存先フォルダの指定と、保存先フォルダを開く操作を追加しました。
- マウスカーソル表示を、より実際のカーソルに近い形状へ変更しました。
- fvCapture用のアプリアイコンを追加し、ウィンドウとWindows実行ファイルに設定しました。
