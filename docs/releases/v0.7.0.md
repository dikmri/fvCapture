# v0.7.0 リリースノート

- アプリ起動時のウィンドウ位置と初期サイズを調整し、画面からはみ出しにくい状態で起動するようにしました。
- 前回利用した録画範囲、操作ラベル、出力先、言語などの設定を次回起動時に復元するようにしました。
- 出力形式、出力サイズ、保存先の指定を録画後プレビュー画面へ移動しました。
- 録画後プレビューで選択した出力設定を使って、選択範囲だけを書き出せるようにしました。
- アプリフォント変更機能を削除し、同梱フォントを固定利用するシンプルな表示設定にしました。
- 待機中、録画中、一時停止中でウィンドウアイコンが変わるようにしました。録画中は赤いバッジ、一時停止中は一時停止バッジを表示します。
- 画面が狭い環境でも設定項目を操作しやすいよう、メイン画面を縦スクロールできるようにしました。
